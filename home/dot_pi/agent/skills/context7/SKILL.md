# Context7

Use this skill whenever the user needs current library documentation or examples.

## When to use
- Looking up APIs
- Finding version-specific docs
- Getting usage examples for libraries/frameworks
- Verifying current best practices from docs

## Workflow
1. Identify the library/framework and version if relevant.
2. Query Context7 for the latest docs and examples.
3. Prefer official, source-backed snippets over secondary explanations.
4. Summarize the relevant API and show a small usage example.
5. Mention version caveats if docs differ across releases.

## Output expectations
- Be concise
- Prefer code examples
- Note source/version if available
- Avoid guessing when docs are unclear

## If Context7 is unavailable
Fall back to official documentation or repository source.
